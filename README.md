# Órbita Rural

## Tema
Satélites e apps para orientação ao produtor rural

## Objetivo
Mostrar como imagens de satélite e aplicativos agrícolas ajudam produtores rurais a monitorar áreas produtivas, recursos naturais e riscos ambientais para tomar decisões mais eficientes e sustentáveis.

## Descrição
O site foi construído como uma plataforma tecnológica de monitoramento, não como uma landing page tradicional. A experiência principal é a **Central Órbita Rural**, onde o usuário escolhe região, cultivo e clima para gerar um relatório dinâmico com condição da vegetação, irrigação, risco ambiental, recomendação e preservação.

## Funcionalidades
- Abertura com composição visual Satélite -> Dados -> Aplicativo -> Produtor;
- Fluxo explicando como os dados orbitais viram decisões no campo;
- Central Órbita Rural com relatório gerado por JavaScript;
- Mapa interativo da propriedade com áreas clicáveis;
- Alertas inteligentes atualizados conforme análise;
- Camadas de satélite ligáveis: vegetação, água, temperatura, solo e uso da terra;
- Indicadores de produção e preservação;
- Menu mobile e recursos de acessibilidade.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript puro

## Como executar
Abra o arquivo `index.html` no navegador.

## Responsividade
O layout usa media queries para adaptar painéis, mapas, fluxo e navegação em telas de computador, tablet e celular.

## Acessibilidade
O projeto inclui foco visível, contraste alto, ajuste de fonte, botão de voltar ao topo, `aria-live` em áreas dinâmicas, rótulos em formulários, textos acessíveis e navegação por teclado.
